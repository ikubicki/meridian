-- phpBB4 "Meridian" Rewrite --

This is a ground-up rewrite of the phpBB forum software.
The original phpBB codebase has been retired. What remains from the
legacy system is the database schema and parts of the data-access logic
tied to it. Everything else — the HTTP layer, authentication, REST API,
frontend, service architecture — has been written from scratch.

Codename:                Meridian

Rewrite Author:          Irek Kubicki <phpbb@codebuilders.pl>
                         https://codebuilders.pl

Tech stack:              PHP 8.4+, Symfony 8.x, React SPA, REST API

What was kept from phpBB3:
  - MariaDB/MySQL database schema (phpbb_* tables)
  - Parts of the DB-access and forum-structure logic
  - Compatible user, forum, topic, post data model

What was rewritten from scratch:
  - HTTP kernel & routing (Symfony 8.x FrameworkBundle)
  - Authentication (JWT Bearer, unified auth service)
  - REST API (OpenAPI-specified, versioned /api/v1/)
  - Frontend (React SPA consuming the REST API)
  - Dependency injection (Symfony DI container)
  - Test infrastructure (PHPUnit 10, Playwright E2E)
  - Docker / deployment configuration
